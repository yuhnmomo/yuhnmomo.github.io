## 清羽傳 - 妖界地圖

### 荒野密林區域
- **荒野密林入口**: 森林的邊緣，光線尚足，能聽到遠處傳來的獸吼。空氣中彌漫著潮濕的土腥味和草木的清新，是人間與妖界氣息的交界處。
    - **出口**:
        - 向外 -> 洛水官道
        - 深入 -> 狼之疆域

- **狼之疆域**: 這片區域的樹木變得異常密集，地上滿是巨大野獸的足跡。空氣中隱隱有著一股孤傲、警惕的野性氣息，彷彿正被無數雙眼睛在暗中窺伺。
    - **棲息角色**: 蒼影 (狼妖)
    - **邂逅條件**: 建議境界2(化形境)以上。
    - **出口**:
        - 向外 -> 荒野密林入口
        - 向東 -> 猛虎巢穴
        - 向南 -> 沼澤邊緣

- **猛虎巢穴**: 一處地勢較高的山岩地帶，巨大的岩石上佈滿了深刻的利爪劃痕。空氣中充滿了霸道、不容侵犯的王者氣息，讓人呼吸都感到一絲壓迫。
    - **棲息角色**: 焰炎 (虎妖)
    - **邂逅條件**: 建議境界2(化形境)以上。
    - **出口**:
        - 向西 -> 狼之疆域

- **沼澤邊緣**: 森林南部的低窪地帶，空氣變得濕熱而黏膩。白色的瘴氣如紗般在林間飄蕩，前方是更危險、能見度更低的沼澤深處。
    - **棲息角色**: 玄麟 (蛇妖)
    - **邂逅條件**: 建議境界3(通玄境)以上。
    - **出口**:
        - 向北 -> 狼之疆域
        - 深入 -> 玄水洞

- **玄水洞**: 洞口隱藏在一片巨大的蕨類植物與垂下的苔蘚之後，極為隱蔽。洞內潮濕陰冷，四壁濕滑，但並不黑暗。牆壁上生長著會發出幽幽藍光的奇異苔蘚，將洞穴映照得如同水下龍宮。洞穴深處有一方水潭，潭水漆黑如墨，深不見底，據說直通地脈水眼，那便是玄麟的寢宮。
    - **棲息角色**: 玄麟 (蛇妖)
    - **邂逅條件**: 通常需要玄麟的允許或引導才能進入。
    - **出口**:
        - 向外 -> 沼澤邊緣

### 隱藏區域
- **毒之花園**: (隱藏) 一片被巨大荊棘與彩色毒藤包圍的山谷。谷內盛開著各種妖異而美麗的巨大花朵，空氣中瀰漫著甜膩誘人的香氣，美麗又致命。
    - **棲息角色**: 華 (曼陀羅花妖)
    - **邂逅條件**: 建議境界4(洞虛境)以上，且需特殊機緣。
    - **出口**: (從特定地點進入)

- **扶桑古樹**: (隱藏) 需穿過一片迷霧幻境才能到達的林中空地。一顆巨大、散發著溫暖光芒的古樹靜靜矗立，空氣中充滿了純粹的生命氣息，令人心神寧靜。化為人形的樹精（沐春）似乎在守護著通往更深處的道路。
    - **棲息角色**: 沐春 (扶桑樹精)
    - **邂逅條件**: 建議境界5(歸墟境)，且需特殊機緣。
    - **出口**:
        - 向外 -> (返回幻境入口)
        - 深入 -> 白澤書齋 (需通過沐春的考驗)
        - 旁邊 -> 狐居正廳 (需完成沐春的贈予)

### 狐居區域
- **狐居正廳**: 由扶桑神木的根系天然交織而成的一處溫暖洞穴，內部別有洞天。這裡是狐居的中心，藤蔓編織成的桌椅樸實而舒適，牆壁上嵌著的發光靈石提供了柔和的照明。空氣中充滿了草木的清香與溫和的生命靈氣。
    - **棲息角色**: 你，以及所有被邀請的同住者
    - **出口**:
        - 向外 -> 扶桑古樹
        - 東 -> 你的臥房
        - 西 -> 客房
        - 北 -> 靜心根室
        - 南 -> 月泉浴室
        - 東南 -> 膳房

- **你的臥房**: 位於狐居最深處、最安靜的房間。巨大的樹根構成一張舒適的床鋪，上面鋪著由柔軟苔蘚和狐尾草製成的床墊，極為舒適。這裡是你最私密的個人空間。
    - **棲息角色**: 你
    - **出口**:
        - 向西 -> 狐居正廳

- **靜心根室**: 一個與扶桑樹根系直接相連的特殊空間。你能在此處清晰地感受到大地的脈動和生命靈氣的流轉，是絕佳的修煉與冥想之地。
    - **棲息角色**: (無，但適合獨自修煉)
    - **出口**:
        - 向南 -> 狐居正廳

- **客房**: 為被你邀請的同住夥伴所準備的房間，風格簡潔但同樣舒適。每個房間都有獨立的石床和發光靈石，確保夥伴也能有自己的私密空間。
    - **棲息角色**: 被邀請的同住者
    - **出口**:
        - 向東 -> 狐居正廳

- **月泉浴室**: 引入了扶桑樹淨化的靈泉而形成的小型溫泉池。池水清澈，蘊含著溫和的治癒力量，能洗去一身的疲憊與塵埃。
    - **棲息角色**: (無)
    - **出口**:
        - 向北 -> 狐居正廳

- **膳房**: 一個小巧而潔淨的膳房。這裡沒有爐火，只有一個由樹根盤繞而成的石台，可以利用扶桑樹的溫和靈力來加熱食物或熬煮藥膳。儲物架上放著一些採集來的仙果與露水。
    - **棲息角色**: (無，但可在此處理食材)
    - **出口**:
        - 向西北 -> 狐居正廳

### 南海龍宮區域
- **龍宮入口**: (隱藏) 在南海之濱的特定區域，通過信物或試煉，可以找到通往海底龍宮的結界入口。穿過結界，巨大的水下宮殿群映入眼簾，華麗非凡。
    - **邂逅條件**: 建議境界4(洞虛境)以上，在「南海之濱」觸發特殊劇情。
    - **出口**:
        - 向外 -> 南海之濱
        - 向主殿 -> 龍君殿
        - 向側殿 -> 珊瑚居

- **龍君殿**: 由整塊巨大的白色暖玉雕琢而成的宏偉宮殿，是龍君敖淵的正式居所。殿內裝飾著金色的龍紋浮雕和巨大的夜明珠，氣勢磅礴，充滿了屬於上位者的威嚴與高貴。
    - **棲息角色**: 敖淵 (龍族)
    - **出口**:
        - 向外 -> 龍宮入口

- **珊瑚居**: 一座由五光十色的活珊瑚自然生長而成的美麗居所，是鮫人王子滄瀾的住處。這裡沒有牆壁，只有溫柔的水流和穿梭其中的奇異小魚。居所中央有一張由巨大貝殼形成的床鋪，閃爍著珍珠般的光澤。
    - **棲息角色**: 滄瀾 (鮫人)
    - **出口**:
        - 向外 -> 龍宮入口

### 白澤書齋區域
- **白澤書齋**: (隱藏) 古樹之後，是一座完全由白玉和星光構成的宏偉書齋，無數古籍在空中緩緩漂浮、自行翻頁。此處極為安靜，充滿了超然物外的智慧與神聖感。上古神獸白澤正坐於其中，翻閱著一卷無字天書。
    - **棲息角色**: 白澤 (上古神獸)
    - **邂逅條件**: 需通過沐春的考驗。
    - **出口**:
        - 向外 -> 扶桑古樹
