## 清羽傳 - 人間界地圖

### 瑤華宮區域
- **瑤華宮山門**: 雲霧繚繞的白玉山門，被強大結界封鎖。空氣中靈氣濃郁，仙鶴偶爾從雲間飛過，此地不染凡塵，氣氛莊嚴而神聖。
    - **人物**: (無)
    - **出口**:
        - 上山 -> 瑤華正道

- **瑤華正道**: 千年寒玉鋪成的小徑，蜿蜒向上。道路兩旁是精心修剪的靈植，散發著淡淡清香。此處是宮內主要通道，偶爾能見到行色匆匆的同門。
    - **人物**: 蕭逸, 蓉蓉
    - **出口**:
        - 北 -> 無月殿
        - 東 -> 演武坪
        - 西 -> 玉清迴廊

- **無月殿**: 師父雲澈的居所。殿內空曠清冷，幾乎沒有任何多餘的裝飾，只有一架古琴和一張軟榻。空氣中瀰漫著非人間草木的冷香，腳步聲在光潔如鏡的地面上會產生清晰的迴響，氛圍莊嚴而寂靜。
    - **人物**: 雲澈
    - **出口**:
        - 南 -> 瑤華正道

- **演武坪**: 開闊的白石廣場，地面佈滿深淺不一的劍痕。此處充滿了陽剛與活力的氣息，兵刃交擊之聲和師兄們的呼喝聲不絕於耳。
    - **人物**: 蕭逸, 韓江
    - **出口**:
        - 西 -> 瑤華正道
        - 東 -> 後山竹林
        - 北 -> 望月臺

- **望月臺**: 由整塊寒玉雕琢而成的高臺，地勢開闊，是宮中觀賞雲海與星辰的最佳地點。夜間月華如水，灑滿平台，靈氣格外充沛，適合靜坐吐納。
    - **人物**: 雲澈, 墨塵
    - **出口**:
        - 南 -> 演武坪

- **後山竹林**: 一片幽靜的紫色竹林，風吹過時，竹葉發出沙沙的聲響。林間光影斑駁，氣氛靜謐，適合冥想，但也可能藏著不為人知的秘密。
    - **人物**: 墨塵
    - **出口**:
        - 西 -> 演武坪
        - 深處 -> 合歡池

- **合歡池**: 隱藏在竹林深處的露天溫泉，終年被溫暖的霧氣籠罩。池水由地脈靈泉匯聚而成，有洗滌經脈、舒緩疲勞之效，是宮中弟子私下放鬆的去處。
    - **人物**: 墨塵, 蕭逸, 如月
    - **出口**:
        - 向外 -> 後山竹林
        - 再深入 -> 寒冰洞

- **寒冰洞**: 位於後山最深處的天然冰洞，洞內寒氣刺骨，萬年不化。洞壁光滑如鏡，倒映著幽藍的寒光，是宮中弟子面壁思過或修煉特殊功法的地方。
    - **人物**: 雲澈, 韓江
    - **出口**:
        - 向外 -> 合歡池

- **玉清迴廊**: 連接弟子居所的長廊，廊外是精心打理的庭院。此處光線柔和，能聞到從丹房飄來的藥草香，或是從各個房間傳出的淡淡熏香。
    - **人物**: 如月, 墨塵
    - **出口**:
        - 東 -> 瑤華正道
        - 南 -> 你的房間
        - 北 -> 丹房藥圃
        - 西 -> 膳房

- **膳房**: 瑤華宮準備餐食的地方。此處不見尋常廚房的油煙，只有處理靈谷、仙果和藥膳的清香。一切都井井有條，潔淨如新。
    - **人物**: 如月, 蕭逸
    - **出口**:
        - 東 -> 玉清迴廊

- **你的房間**: 屬於你的私人空間，佈置簡潔雅致，有一扇面向庭院的窗戶。這裡是你唯一可以完全放鬆、不受打擾的地方。
    - **人物**: (無)
    - **出口**:
        - 出門 -> 玉清迴廊

- **丹房藥圃**: 如月師姊照料的藥圃，空氣中始終瀰漫著濃郁的藥草香氣。數百種珍稀草藥在靈氣滋養下生機勃勃，幾隻藥蝶在花叢間飛舞。
    - **人物**: 如月
    - **出口**:
        - 南 -> 玉清迴廊

- **藏書閣**: 收藏了無數孤本秘籍的書閣。閣內極為安靜，只有翻動書頁的聲音。空氣中充滿了墨香、舊紙張和防蟲草藥混合的獨特氣味。
    - **人物**: 韓江
    - **出口**: (從玉清迴廊進入)

- **禁地靈泉**: 被強大陣法封鎖的宮內禁地。靈氣在此處化為肉眼可見的白色霧氣，中央是一池不斷散發著寒氣的靈泉，是突破境界的修煉聖地。
    - **人物**: 雲澈
    - **出口**: (無)

### 山下世界 - 過渡區域
- **瑤華山腳**: 仙山的霧氣在此處散去，紅塵俗世的氣息撲面而來。一條黃土官道向遠方延伸，連接起凡人的世界。
    - **出口**:
        - 東南 -> 洛水官道

- **洛水官道**: 通往洛水鎮的黃土路，路面因車馬碾壓而略顯顛簸。路邊有座荒廢的破廟，偶爾能見到趕路的商旅或江湖人士。
    - **出口**:
        - 西北 -> 瑤華山腳
        - 東南 -> 洛水鎮鎮口
        - 南 -> 荒野密林
        - 東北 -> 天京官道

- **荒野密林**: 人跡罕至的原始森林。林中光線昏暗，古木參天，空氣潮濕，充滿了泥土、腐葉和野獸的氣息，危險與機遇並存。
    - **人物**: 段浪 (沈焰陽)
    - **出口**:
        - 北 -> 洛水官道
        - 深處 -> 迷霧沼澤
        - 東 -> 青丘古道
        - 南 -> 南海之濱

- **迷霧沼澤**: 終年被白色瘴氣籠罩的沼澤，能見度極低。空氣濕熱而黏膩，腳下是深淺不知的泥潭，四處傳來毒蟲的嘶鳴，充滿了未知與兇險。
    - **人物**: 秦越
    - **出口**:
        - 外圍 -> 荒野密林

- **天京官道**: 由平整的青石板鋪成的寬闊大道，直通帝都。路上行人與車馬絡繹不絕，氣氛比尋常官道更為肅穆，時有佩刀的官兵巡邏。
    - **出口**:
        - 西南 -> 洛水官道
        - 東北 -> 天京城，朱雀門

- **南海之濱**: 濕鹹的海風吹拂著臉頰，眼前是無垠的碧藍大海。柔軟的沙灘上散落著奇異的貝殼，海浪拍打著礁石，發出陣陣轟鳴。
    - **出口**:
        - 向北 -> 荒野密林

### 洛水鎮區域
- **洛水鎮**: 一座繁華的邊境城鎮，鎮口的木製牌坊顯得古樸。這裡魚龍混雜，既有凡人商販，也有修真者往來，充滿了江湖氣息。
    - **出口**:
        - 西北 -> 洛水官道
        - 進鎮 -> 朱雀大街

- **朱雀大街**: 洛水鎮最繁華的街道，人聲鼎沸，商舖林立。小販的叫賣聲、孩童的嬉鬧聲、酒樓的絲竹聲交織在一起，充滿了活色生香的人間煙火氣。
    - **人物**: 謝安之 (謝允)
    - **出口**:
        - 東 -> 東市
        - 西 -> 悅來客棧
        - 北 -> 鎮衙門
        - 南 -> 洛水河畔

- **東市**: 充滿叫賣聲的露天市集，空氣中混合著瓜果、香料和汗水的味道。人群擁擠，需要時刻注意自己的財物。
    - **出口**:
        - 西 -> 朱雀大街
        - 南 -> 回春堂

- **悅來客棧**: 鎮上最大的客棧，大堂裡人聲嘈雜，說書先生正講到精彩處。空氣中混雜著酒菜的香氣和旅人的風塵僕僕。
    - **出口**:
        - 出門 -> 朱雀大街

- **鎮衙門**: 官府之地，門前有兩座石獅，衙役按刀站崗，氣氛嚴肅，周圍的百姓路過時都會下意識地放輕腳步。
    - **出口**:
        - 南 -> 朱雀大街

- **洛水河畔**: 清澈的河流穿城而過，岸邊栽滿了隨風搖曳的柳樹。此處比主街安靜許多，能聽到潺潺的水流聲和鳥鳴，是個鬧中取靜的好去處。
    - **人物**: 謝安之 (謝允)
    - **出口**:
        - 北 -> 朱雀大街

- **回春堂**: 隱藏在僻靜小巷深處的藥鋪，門口掛著褪色的幌子。與巷外的喧囂不同，這裡異常安靜，空氣中飄散著濃郁而複雜的藥草味。
    - **人物**: 秦越
    - **出口**:
        - 北 -> 東市

### 都城天京區域
- **天京城**: 高達數十丈的宏偉城都，給人帶來巨大的壓迫感。城牆下，守衛森嚴，盤查嚴格，彰顯著天子腳下的威嚴與秩序。
    - **出口**:
        - 出城 -> 天京官道
        - 進城 -> 朱雀大街

- **朱雀大街 (天京)**: 貫穿天京南北的主幹道，街道由潔白的玉石鋪就，兩旁是雕樑畫棟的樓閣。與洛水鎮不同，這裡的繁華中帶著一種莊嚴與肅穆。
    - **出口**:
        - 南 -> 天京城，朱雀門
        - 北 -> 皇城承天門
        - 東 -> 東市琉璃坊
        - 西 -> 西市太白樓

- **皇城承天門**: 皇城的正門，金瓦紅牆，氣勢恢宏。門前是巨大的廣場，禁軍林立，氣氛莊嚴肅殺，禁止普通百姓靠近。
    - **人物**: 慕容璟
    - **出口**:
        - 南 -> 朱雀大街
        - 東 -> 國師府

- **國師府入口**: 皇城之內，一座氣勢恢宏的府邸。門前有兩座巨大的麒麟石雕，重兵把守，氣氛森嚴，無形的壓力讓人不敢直視。
    - **人物**: 國師府守衛
    - **出口**:
        - 向外 -> 皇城承天門
        - 進入 -> 天權殿

- **天權殿**: 國師府的正殿，氣勢恢宏。黑曜石鋪成的地面光可鑑人，巨大的樑柱上雕刻著星辰運行的圖紋。空氣中瀰漫著權力與檀香的氣息，令人不自覺地屏住呼吸。
    - **人物**: 玄皓
    - **出口**:
        - 向外 -> 國師府入口
        - 後堂 -> 觀星樓
        - 側廊 -> 寢居

- **觀星樓**: 位於國師府最高處的塔樓，是玄皓的書房。這裡收藏著無數來自各地的孤本與星象圖。一架巨大的青銅觀星儀佔據了房間的中央，緩緩轉動，似乎在推演著天下所有人的命運。
    - **人物**: 玄皓
    - **出口**:
        - 向下 -> 天權殿
        - **(需觸發機關) -> 鎖魂堂**

- **鎖魂堂**: (隱藏) 隱藏在觀星樓書架後的密室。牆壁上刻滿了用於禁錮和抽取靈魂的血色符文。房間中央的石台上，擺放著從青丘奪來的聖物和數個裝著妖族精魄的容器，散發著不祥的氣息。
    - **人物**: (通常無人，但守衛森嚴)
    - **出口**:
        - 向外 -> 觀星樓

- **寢居**: 玄皓私人的休息場所。房間的佈置意外地簡潔，甚至可以說是樸素，只有一張由整塊寒玉雕琢而成的**石床**和一個蒲團，用以獨自冥想與休息，象徵著他極致的自律與清冷。然而，在房間的角落，卻收著一張毛皮極為厚實柔軟的**雪狼皮毛地毯**，似乎只在極特殊的時刻才會被鋪陳在房間中央的陣法之上。
    - **人物**: 玄皓
    - **出口**:
        - 向外 -> 天權殿

- **東市琉璃坊**: 經營絲綢珠寶的奢華商業區，空氣中都彷彿飄著金錢與脂粉的香氣。這裡的建築更為精緻華美，往來皆是達官貴人。
    - **人物**: 墨塵
    - **出口**:
        - 西 -> 朱雀大街
        - 深處 -> 鬼市入口

- **鬼市**: 位於偏僻死胡同中，只在夜晚出現的黑市。這裡沒有光亮，交易全在黑暗中進行，空氣中充滿了秘密、危險和一絲血腥味。
    - **人物**: 段浪 (沈焰陽)
    - **出口**: (夜晚限定)

- **西市太白樓**: 平民聚集的西市和最著名的九層酒樓。這裡比東市更具市井氣息，三教九流匯聚，人聲嘈雜，是打探消息的絕佳場所。
    - **出口**:
        - 東 -> 朱雀大街
        - 北 -> 文曲坊

- **文曲坊**: 天京城的文化中心，街道兩旁是林立的書齋和筆墨鋪。空氣中瀰漫著濃郁的書香氣息，氛圍比其他街區要安靜儒雅得多。
    - **人物**: 謝安之 (謝允)
    - **出口**:
        - 南 -> 西市太白樓
        - 東 -> 七皇子別院

- **七皇子別院**: 一座極其幽靜的院落，高牆將內外的喧囂隔絕。院內種滿了藥草，空氣中飄著淡淡的苦澀藥味，氛圍寧靜而壓抑。
    - **人物**: 慕容璟
    - **出口**:
        - 西 -> 文曲坊

- **太醫院**: 專為皇室看診的機構，防衛森嚴。院內瀰漫著極為珍貴的藥材香味，但氣氛卻因與皇權掛鉤而顯得格外緊張肅穆。
    - **人物**: 秦越, 慕容璟
    - **出口**: (需特殊途徑進入)

### 青丘決戰區域 (最終章)
- **青丘古道**: 一條被世人遺忘的古老小徑，空氣中充滿了悲傷的氣息。道路兩旁的植物都呈現出一種不祥的灰敗之色，狐族殘留的幻術讓周圍的景象時而扭曲。
    - **出口**:
        - 西 -> 荒野密林
        - 盡頭 -> 青丘結界

- **青丘結界**: 破碎的巨大結界懸浮在空中，殘存的碎片散發著不祥的紫光。空氣中的靈力紊亂而狂暴，靠近時能感受到刺骨的悲鳴。
    - **出口**:
        - 進入 -> 昔日之墟

- **昔日之墟**: 你的故鄉，如今只剩下斷壁殘垣。空氣中瀰漫著燒焦的氣味和揮之不去的血腥。風吹過空無一人的街道，發出嗚咽般的聲響，悲傷得令人窒息。
    - **人物**: (敵方巡邏兵)
    - **出口**:
        - 深處 -> 帝女宮前

- **帝女宮**: 曾經的青丘主殿，你的家。如今宮殿的大門緊閉，上面刻滿了不屬於青丘的符文，門前守衛森嚴，充滿了肅殺之氣。
    - **人物**: (敵方精銳)
    - **出口**:
        - 闖入 -> 九尾天狐殿

- **九尾天狐殿**: 莊嚴的主殿已被玷污，四處懸掛著敵人的旗幟。你的仇人正高坐在本該屬於你母親的王座之上，空氣中滿是侵略者的得意與傲慢。
    - **人物**: (最終BOSS)
    - **出口**:
        - 後殿 -> 枯萎聖樹

- **枯萎聖樹**: 青丘的靈脈核心，一棵巨大的古樹，如今已經焦黑枯萎，了無生機。這裡是整個青丘最悲傷的地方，連風都靜止了。這裡是故事的終點，也是決定命運的抉擇之地。
    - **出口**: (無)
